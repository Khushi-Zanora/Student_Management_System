# Student Performance Management System
A Django-based web application to manage students and track academic performance.

# Features
- Student detail cards
- Add and manage test marks
- Total and average score calculation
- Visual progress bar for performance
- Clean and simple UI

# Tech Stack
- Django (Python)
- HTML, CSS
- SQLite
- GitHub

# Learning Outcomes
- Django models & views
- Template rendering
- Aggregations in Django ORM
- UI styling and layout

# Future Enhancements
- Authentication system
- Charts for analytics
- MySQL integration
